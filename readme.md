# page-loader-async
Utility for download of files from the web.

### Usige
```sh
$ npm i -g page-loader-async
$ page-loader-async [options] <url>
```

### Options
```
-h, --help           output usage information
-V, --version        output the version number
-o, --output [path]  output path
```
